This project was created as part of the 42 curriculum by pneto-vi.

It is a library of functions bundled together for practical use in future projects. The project can be tested using the provided Makefile. Using the make command compiles the source code into a .a static library with the object files integrated.

The Makefile also includes the following rules:
	
	clean: Removes all object files.

    fclean: Removes all object files and the compiled .a library.

    re: Performs a full recompilation (fclean + make).
	
Regarding resources, I primarily used man pages for function. Beyond the header and Makefile, I conducted research in stages to build the library incrementally. I utilized AI to understand new concepts, but followed a practice of removing the assisted code and rewriting it from the beggining. I also used technical blogs and AI to understand how to properly structure the project testers.
